# Aplikasi To-Do List Sederhana

## Deskripsi Program

Aplikasi To-Do List Sederhana berbasis CLI ini adalah program pengelola tugas yang dikembangkan menggunakan Python. Program memungkinkan pengguna menambah tugas baru, melihat daftar tugas, menandai tugas selesai, menghapus tugas, memfilter berdasarkan status, dan melihat statistik progress. Data disimpan menggunakan List dan Dictionary tanpa memerlukan database eksternal.

**Fitur Utama:**
- Lihat Semua Tugas
- Tambah Tugas Baru
- Tandai Tugas Selesai
- Hapus Tugas
- Filter Tugas (Belum/Selesai)
- Lihat Statistik Progress

## Cara Menjalankan Program

1. Pastikan Python 3.x sudah terinstal di komputer
2. Simpan file `todolist.py` di folder kerja Anda
3. Buka terminal atau command prompt
4. Arahkan ke folder tempat file `todolist.py` berada:
   ```
   cd path/to/folder
   ```
5. Jalankan program dengan perintah:
   ```
   python todolist.py
   ```
6. Pilih menu yang diinginkan (1-7)

**Catatan:** Data akan hilang setelah program ditutup karena menggunakan penyimpanan sementara.
